On this page, the user could update their profile information, password, check leaves, and could be able to log out of the system.

1. **How to update the user profile information ?**
   
      - On the menu, click the **‘user name’** and then the **‘Update Profile’** sub-menu.
      - The **‘Update Profile’** page is displayed.
      - Review and update the profile information that you wish to change on the **‘Update Profile’** page and click the **‘Save Changes’** button.
      - Upon updating the profile information that you wish to change and clicking the **‘Save Changes’** button on the **‘Update Profile’** page, a successful confirmation message is displayed. 

2. **How to update the user password ?**
   
      - On the menu, click the **‘user name’** and then the **‘Change Password’** sub-menu.
      - The **‘Change Password’** page is displayed.
      - Fill out the **‘New Password’** and **‘Confirm New Password’** text fields and click the **‘Update Password’** button.
      - Upon entering the **‘New Password’** and clicking the **‘Update Password’** button on the **‘Change Password’** page, a successful confirmation message is displayed.
